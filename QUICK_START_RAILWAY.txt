📋 CHECKLIST - Follow in Order:

□ 1. Login to Railway (https://railway.app)
     → Click "Login with GitHub"
     → Authorize Railway

□ 2. Create New Project
     → Click "New Project"
     → Select "Deploy from GitHub repo"
     → Choose: "abhiishekbisht/Marksheet-Generator"
     → Click "Deploy Now"

□ 3. Add MySQL Database
     → Click "+ New" in project
     → Select "Database" → "Add MySQL"
     → Wait 30 seconds

□ 4. Add Environment Variables
     → Click Flask service → "Variables" tab
     → Add these variables:
     
     SECRET_KEY=gulzar-marksheet-secret-2024-railway
     FLASK_ENV=production
     FLASK_DEBUG=False
     COLLEGE_NAME=GULZAR GROUP OF INSTITUTIONS
     COLLEGE_ADDRESS=Academic Excellence Since 1995
     
     (Database vars are auto-configured!)

□ 5. Generate Domain
     → Flask service → "Settings"
     → Scroll to "Networking"
     → Click "Generate Domain"
     → Copy your URL!

□ 6. Initialize Database
     Option A - CLI:
       npm install -g @railway/cli
       railway login
       railway link
       railway run python init_db.py
     
     Option B - Shell:
       Flask service → Settings → Shell
       Run: python init_db.py

□ 7. Test Your App!
     → Visit your Railway URL
     → Login: admin / admin123
     → Test marksheet generation
